The system simulates how a computer manages storage, memory, applications, files, and user settings behind the scenes while providing a simple interface for end users. (using core OOP concepts)

## Features
### System Components
- Storage (Hard Disk) management
- RAM allocation and deallocation
- Application lifecycle management
- Computer system status monitoring

### Applications
#### File Explorer
- Create files
- View files
- Delete files
- List stored files

#### Recycle Bin
- Recover deleted files
- Permanently delete files
- Empty recycle bin
- Dynamic file management

#### Calculator
- Basic arithmetic operations
- Power and square root calculations
- Variable storage (A-Z)
- Statistical operations
- Calculation history tracking
- Export history to storage

#### Settings
- Theme customization
- Font settings
- Language preferences
- Date format configuration
- Reset to default settings
